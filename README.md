# A Digital Clockwork

A **Digital Clockwork** é um simulador de um circuito de relógio digital inspirado no projeto de Wagner Rambo, apresentado em seu canal **WR Kits**.

Este simulador foi criado para estudar e aprender sobre simulação de circuitos digitais, conceitos de baixo nível em hardware e lógica discreta, e o funcionamento interno de chips como **4017**, **4029**, **4511** e entre outros chips que o Wagner Rambo usou para a construção do seu próprio relógio digital.

Durante minhas férias da faculdade, este repositório servirá como meu laboratório experimental para explorar hardware, programação em  C++ e simulação de circuitos digitais por conta própria.
